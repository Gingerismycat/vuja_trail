# vuja_trail

By: Vuk Arula (2023)

## Installation Guide
1. Create a Conda environment
    ```console
    conda create --name vuja-trail python=3.10
    ```
2. Activate the Environment
    ```console
    conda activate vuja-trail
    ```

3. To run/play the VUJA-TRAIL call 
    ```console
    python main.py
    ```
